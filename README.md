# MFV Encryption and decryption
> This is the official Repository of MFV encryption
## Usage
### Encryption
`crypt.py` 
  - `-e` # Encrypt
  - `-t 'some text'` # text you want to encrypt
  - `-k 12` # key you want to use

### Decryption
`crypt.py` 
  - `-d` # Decrypt
  - `-t '123 456 789'` # ints you want to decrypt
  - `-k 12` # key you want to use
